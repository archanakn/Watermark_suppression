# Watermark_suppression
NOTICE:- This code is related to the manuscript titled "Enhancing Dermatological Image Classification via Luminance-Guided Autoencoder Watermark Suppression" and is currently submitted to THE Visual Computer for review.


A MATLAB code for the suppression of watermarks in medical images
The code is designed to suppress text watermarks, which can obscure edges. 
Not removal, but will truncate the edges of the watermark. Best for the Dermnet dataset, where the watermark is 86% transparent.
DOI[https://doi.org/10.5281/zenodo.18062017]
